# VIES + BCE + PEPPOL Checker — Dental Addict

## Déploiement Render (gratuit)

1. Crée un compte sur https://render.com
2. New → Web Service → Connect a repository
3. Connecte ton GitHub et pousse ce dossier dedans
4. Render détecte automatiquement Python
5. Start command : `python server.py`
6. Deploy → URL disponible en 2 minutes

## Usage local (Mac)

```bash
cd dental_checker
python3 server.py
```

Ouvre http://localhost:7847
